This project shows how different feature scaling techniques affect the distribution of data using a real world dataset .\
The dataset was downloaded from kaggle.\
**Observations :**\
The difference in distribution of original data and the distributions of scaled data can be very clearly seen in comparison_graph.png \
The difference of distibution of original data disrtibution and disrtibution of scaled data shows that our machine learning model would become inaccuarte for this particular dataset if not scaled properly . 

Also , there is difference in the distribution of differently scaled datasets especially when you notice the tails.\
The tails of various features in differently scaled datsets become more distinictively visible as you see from min-max scaling -> z -score scaling -> robust scaling .\
